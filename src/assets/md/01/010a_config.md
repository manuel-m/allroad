###Setup

1. Unarchive **`allroad_template.bz2`**
2. Put your **markdown** files into **`md`** and edit **`TheContent.json`**
```
├── assets
│   ├── images
│   │   └── knownissues.png
│   ├── md
│   │   ├── 00
│   │   │   ├── 000_allroad.md
│   │   │   └── 001_features.md
│   │   └── 01
│   │       ├── 010_config.md
│   │       ├── 011_protection.md
│   │       └── 012_knownissues.md
│   └── TheContent.json
├── favicon.ico
├── fonts
├── index.html
├── scripts
└── styles
```
